# YSTV

Yonsei TV

# 실행방법

[`pnpm`](https://pnpm.io/) 패키지 매니저를 사용합니다

### 개발

```bash
pnpm i
pnpm dev
```

## 소스코드 구조

### Frontend

- `src/app/page.js` - React page component
- `src/components/hsl.jsx` - React HSL Component
